Regex Generator

This tool is designed to aid with the genration and testing of regualar expressions, the current build version is 0.0.1 Alpha. This tool will allow you to add a test string in the top of the content along with some control data once you click escape, the tool will automatically escape strings, so that they are ready for use with regular expressions. Following which you can still append or add test data to the "test data" before makeing modications to the regex, once any changes have been detected, the resuls section will come into view, where you can make as many changes untill you are happy with the results.
